Introducción a GIT

Estamos empezando a usar GIT.

Podemos agregar o elimiar líneas en nuestro código y GIT tomará los cambios.

Uno de los principales comandos de GIT es el Commit.

Un Commit es una afirmación de un cambio en nuestro código.
